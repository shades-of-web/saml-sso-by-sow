== Description ==
Experience seamless single sign-on (SSO) integration between your IDP and WordPress with our cutting-edge plugin. Eliminate the hassle of multiple logins and streamline user authentication across platforms. Our plugin empowers website owners to effortlessly synchronize user credentials, ensuring a smooth and secure login experience for both administrators and users. Say goodbye to login fatigue and embrace the simplicity of unified authentication with \"SAML SSO by SoW\" plugin for WordPress.

== Installation ==
1. Install the plugin from the WordPress plugin repository.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->SAML SSO by SoW screen to configure the plugin.
4. In case of multisite, use menu on the network admin dashboard called "SAML SSO by SoW" to configure the plugin.

== Frequently Asked Questions ==

= What is SAML? =

SAML (Security Assertion Markup Language) is an open standard for exchanging authentication and authorization data between parties, in particular, between an identity provider and a service provider.

= Is this plugin compatible with all IDPs? =

While \"SAML SSO by SoW\" is designed to be compatible with a wide range of IDPs, specific configurations might vary. Please consult your IDP documentation for compatibility and setup details.

= Can I use this plugin in a multisite environment? =

Yes, \"SAML SSO by SoW\" supports multisite environments. Network activation is possible, and individual site settings can be configured if needed.
